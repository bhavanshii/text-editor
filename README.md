# 📝 Simple Text Editor (Python - Tkinter)

A basic desktop text editor built using Python and Tkinter. This project allows users to create, open, edit, and save text files with a simple GUI interface.

---

## 🚀 Features

* Create a new file
* Open existing `.txt` files
* Edit text easily
* Save files
* Simple and clean UI
* Pop-up message after saving

---

## 🛠️ Technologies Used

* Python 🐍
* Tkinter (GUI library)

---

## ▶️ How to Run

1. Make sure Python is installed
2. Clone this repository:

   ```bash
   git clone https://github.com/YOUR-USERNAME/text-editor.git
   ```
3. Navigate to the project folder:

   ```bash
   cd text-editor
   ```
4. Run the program:

   ```bash
   python main.py
   ```
